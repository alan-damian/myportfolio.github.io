HTML

Se compone de etiquetas que permiten dar una estructura inicial a una pagina. existen normas, no obligatorias, pero si fundamentales para tener buenas practicas a la hora de planear un proyecto, como lo son el correcto identado de las estiquetas y el uso semantico de las mismas.

PYTHON

Python es un lenguaje de programación de código abierto con un enfoque imperativo, orientado a objetos y de alto nivel. Además, se caracteriza por tener una sintaxis clara y concisa, lo que lo convierte en un lenguaje relativamente fácil de utilizar.
